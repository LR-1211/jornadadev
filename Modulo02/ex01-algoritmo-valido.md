# a. ?Some dois números.?
-Resposta: Algoritmo Inválido (Não).

-Justificativa: A ordem é muito vaga, não possui uma estrutura de ações ordenadas claras, porque não é dito quais números são estes.

# b. ?Leia um número. Se for positivo, mostre ?positivo?. Senão, mostre ?negativo?.?
-Resposta: Algoritmo válido (Sim).

-Justificativa: A ordem é muito clara, com uma estrutura lógica bem direta, uma condição simples que determina o resultado obtido. 

# c. ?Fique repetindo até acabar.?
-Resposta: Algoritmo inválido (Não).

-Justificativa: A ordem é muito vaga e não é bem estruturada, porque não é dito o que deve ser repetido até acabar e muito menos o que sigifiica até acabar, não informa uma condicional de valores lógicos pro termino do cógido, o deixando ambiguo e infinito.

# d. ?Pegue um ingrediente qualquer e cozinhe.?
-Resposta: Algoritmo inválido (Não).

-Justificativa: A ordem é vaga, não possui uma ordem especificada, é um comando muito abrangente que não dita qualquer especificação lógica nem condicional, permitindo qualquer saida aleatórea. Muito Ambiguo
